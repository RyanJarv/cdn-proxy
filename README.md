## Installation

Download to your project directory, add `README.md`, and commit:

```sh
git checkout git@github.com:RhinoSecurityLabs/cdn-proxy.git
cd cdn-proxy
python3 -m venv venv
git add README.md
git commit -m "Use README Boilerplate"
```
